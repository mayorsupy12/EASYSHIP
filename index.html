# EASYSHIP — Careers Portal 🚢 A freight & shipping job board for **EASYSHIP Inc.** (California). Built with Python (Flask) + vanilla JS. Deploy-ready for Render. ## Features - 6 open freight/shipping job roles - Department & job type filters + search - Expandable job detail view - 3-step application flow: 1. Personal info form 2. Identity verification via Google Meet scheduling 3. Return & complete confirmation - Fully mobile-responsive ## 🚀 Deploy to Render ```bash # 1. Push to GitHub git init && git add . && git commit -m "EASYSHIP Careers Portal" git remote add origin https://github.com/YOUR_USERNAME/easyship-jobs.git git push -u origin main ``` Then on **render.com**: - New → **Web Service** → connect repo - **Build:** `pip install -r requirements.txt` - **Start:** `gunicorn app:app --bind 0.0.0.0:$PORT --workers 2` - Done in ~60 seconds ✅ ## Run Locally ```bash pip install -r requirements.txt python app.py # Open http://localhost:5000 ```
